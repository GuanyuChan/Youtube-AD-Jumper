Firstly, download the release file from the "release" folder. 
Then, open your browser and download "Tampermonkey" from the extension store (Edge/Chrome Web Store). 
Next, go to the extension and create a new script. Paste the "Copy this" file into the script, making sure to delete the original words inside the new scirpt. Finally, click :"file" and click on the save button. 
Your setup should now be complete! Please note that bugs may occur.

首先，從「release」資料夾下載發布檔案。
然後，開啟瀏覽器並從擴充商店（Edge/Chrome Web Store）下載「Tampermonkey」。
接下來，轉到擴充功能並建立一個新腳本。 將「複製此」檔案貼到腳本中，確保刪除新腳本中的原始單字。 最後，點擊：“檔案”並點擊“儲存”按鈕。
您的設定現在應該已完成！ 請注意，可能會出現錯誤。

まず、「release」フォルダからリリースファイルをダウンロードします。
次に、ブラウザを開き、拡張機能ストア (Edge/Chrome Web ストア) から「Tampermonkey」をダウンロードします。
次に、拡張機能に移動して新しいスクリプトを作成します。 「これをコピー」ファイルをスクリプトに貼り付けます。新しいスクリプト内の元の単語は必ず削除してください。 最後に、「ファイル」をクリックし、保存ボタンをクリックします。
これでセットアップは完了したはずです。 バグが発生する可能性がありますのでご了承ください。
